# Тестовый документ для отдела 5

Это тестовый документ, который содержит информацию о различных аспектах работы компании.

## Основные разделы

### 1. Приветствие
Привет! Это тестовый документ для проверки работы RAG системы. Здесь содержится информация о том, как работает система поиска и генерации ответов.

### 2. Техническая информация
Система RAG (Retrieval-Augmented Generation) использует векторный поиск для нахождения релевантных документов и генерацию ответов на основе найденного контекста.

### 3. Процедуры
- Загрузка документов в векторную базу данных
- Создание эмбеддингов для текста
- Поиск релевантных фрагментов
- Генерация ответов на основе контекста

### 4. Контакты
Для получения дополнительной информации обращайтесь к администратору системы.

## Заключение
Этот документ предназначен для тестирования функциональности RAG системы и демонстрации возможностей поиска и генерации ответов.
