Привет! Это тестовый документ для проверки работы RAG системы.

Данный документ содержит информацию о том, как работает система Retrieval Augmented Generation (RAG).

RAG - это технология, которая объединяет возможности языковых моделей с поиском по базе знаний. Система работает следующим образом:

1. Пользователь задает вопрос
2. Система ищет релевантные документы в базе знаний
3. Найденные документы используются как контекст для генерации ответа
4. Языковая модель генерирует ответ на основе контекста

Преимущества RAG:
- Более точные ответы на основе актуальной информации
- Возможность работы с большими объемами данных
- Контролируемость источников информации
- Возможность обновления знаний без переобучения модели

В данной системе используется Yandex Cloud API для эмбеддингов и генерации текста, а также Chroma для векторного поиска.
